在修改完成后发现服务器一直报错，后来发现是以为hash加密后的password会超过定义的100字符，所以修改modles:
```commandline
class UserModel(db.Model):
    # ... 其他字段保持不变
    password = db.Column(db.String(255), nullable=False)
    # ... 其他字段保持不变
```
然后
```commandline
flask db migrate -m "Increase password field length"
flask db upgrade
```
